# <해커톤 대회 참가>
1. 꿈나무 카드의 문제점 : 꿈나무 카드의 사용자들의 이용할 수 있는 가맹점 수와 븐포를 확인할 결과 전체 가맹점의 77%가 편의점이었고, 제과점을 포함할 경우 82%로 이로 인해 사용자들이 카드를 사용할 수 있는 곳이 부족하며, 건강, 먹거리, 블균형우려 등 영양적 문제가 제기 되고 있다.

2. 해결 방안 : 사용자의 구매 제품을 분석하여 사용자에게 필요한 영양소가 무엇인지 제공하고 해당되는 음식을 섭취할 수 있는 가맹점을 보여줌으로써 일반 음식점에 대한 가맹점 확대하고, 사용자의 영양불균형을 해소하는 것을 목표로 한다.
